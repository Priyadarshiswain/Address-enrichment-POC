⚡ Cost Optimisation Recommendations

  • Cache validation results — store address → postcode + verdict in a database. Re-queries of the same address cost nothing. DHL repeat shipments to the same address are common.
  • Skip airport/port lookup for known inland regions — if the postcode is > 100 km from any coast, skip the port search entirely. Saves $0.032 per call (~43% of total cost).
  • Lazy-load the map — only load the Google Maps JS SDK when the user actually scrolls to the map. Saves $0.007 per session for users who only care about the postcode.
  • Batch validation via the API — Google Address Validation supports bulk processing at reduced rates for enterprise customers. Contact Google for volume pricing.
  • Use one provider for map + boundary layers — keeping boundary rendering inside Google Maps reduces integration drift and removes the separate postal polygon dependency.
